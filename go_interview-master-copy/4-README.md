# 大厂面经

我们集中了腾讯，阿里，百度，滴滴，字节，B 站 等一线大厂的核心开发，专注分享国内一线大厂面经，让你轻松拿到 offer，直接内推。

## 我们主要做什么 ?

* 真实的大厂面经分享，让你第一时间掌握当前面试方向
* 源码解读，让你不再惧怕任何面试官刁难
* 每日算法，让你轻松通过算法筛选
* 问题答疑，轻松掌握问题精髓
* 架构分享，提升日常的架构设计能力
* 非正式开车，让你随时飚升高速列车
